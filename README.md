# Pocket CoP

**42 CFR Parts 482 & 489 — CMS Conditions of Participation + Provider Agreements**
Offline-first PWA for inpatient psychiatric hospital reference.

## Deploy to GitHub Pages

1. Create a new **public** GitHub repository
2. Upload all files from this folder
3. **Settings → Pages → Deploy from branch → main / (root)**
4. URL: `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Install on Android

1. Open URL in Chrome → three-dot menu → **Add to Home Screen**
2. Open once while online — all files cache for offline use

## Pushing Updates

1. Edit files, upload to GitHub
2. Bump `CACHE_NAME` in `sw.js` (e.g. `pocket-cop-v1.1`)
3. Green **Update now** banner appears on next online open
